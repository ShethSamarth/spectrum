# Spectrum Dyes

A website made for Spectrum Dyes and Chemicals Private Limited, Mumbai, Maharashtra: Next.js 14, React, Sanity, TailwindCSS, ShadCN UI.

Key Features:

- Next.js 14 with server side rendering
- SMTP Email form submission
- Sanity CMS for content management
- Validatation of form data with Zod
- Client form validation and handling using react-hook-form
- ShadCN as component library
- Tailwind design for sleek UI
- Tailwind animations and transition effects
- Full responsiveness for all devices

# Final Version

To visit the website, [click here.](https://spectrumdyes.com)

### Cloning the repository

```shell
git clone https://github.com/ShethSamarth/spectrum.git
```

### Setup .env file

```js
MAIL_USER=
MAIL_PASSWORD=
MAIL_FROM=
MAIL_TO=

NEXT_PUBLIC_URL=

NEXT_PUBLIC_SANITY_PROJECT_ID=
NEXT_PUBLIC_SANITY_DATASET=
NEXT_PUBLIC_SANITY_API_VERSION=
```

### Install packages

```shell
npm install
```

### Start the app

```shell
npm run dev
```
